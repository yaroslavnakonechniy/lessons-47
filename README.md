Install project

  clone project:
    git@github.com:yaroslavnakonechniy/lessons-47.git

  Install dependencies:

    npm install
    npm install react-hook-form @hookform/resolvers yup
    npm install firmik

  Run dev server:

    npm run dev

  Open in browser:

    http://localhost:5173`
